# AIS Trading Inc Website

Official website for AIS Trading Inc - Premium North American Lumber Exporters

## 🚀 Tech Stack

- **React 18** - Modern UI library
- **Vite 5** - Fast build tool
- **CSS3** - Custom styling with animations

## 🛠️ Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
